# Build an Orchestrator in Go - Educational Website

A modern, interactive website companion for "Build an Orchestrator in Go (From Scratch)" by Tim Boring. This educational platform helps developers visualize and explore container orchestration concepts through an engaging, animated interface.

## Overview

This Next.js application showcases the complete journey of building Babosa, a container orchestrator, from the ground up. It features:

- **Hero Section**: Book introduction with quick facts
- **Stages Section**: Interactive timeline of the 4 parts and 11 chapters
- **Discoveries Section**: Key learnings from building Babosa
- **Challenges Section**: Common pitfalls and solutions
- **FAQ Section**: Comprehensive Q&A
- **Responsive Design**: Fully mobile-friendly
- **Dark Mode**: Kubernetes-inspired dark theme (default)
